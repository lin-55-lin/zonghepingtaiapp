# AI软件集成平台

一个现代化的AI软件管理平台，支持分类管理、搜索、批量操作等功能。

## 功能特性

- 🎯 **智能分类**：支持自定义分类管理
- 🔍 **快速搜索**：实时搜索AI软件
- 🛠️ **批量操作**：支持批量导入、导出、删除
- 🎨 **个性化布局**：可自定义卡片布局和壁纸
- 📱 **响应式设计**：适配各种屏幕尺寸

## 部署说明

### GitHub Pages 部署

1. 将项目上传到GitHub仓库
2. 在仓库设置中开启GitHub Pages
3. 选择部署分支（通常是main或gh-pages）
4. 访问 `https://用户名.github.io/仓库名`

### 本地运行

直接打开 `综合平台.html` 文件即可在浏览器中运行。

## 使用说明

### 基本操作
- **添加卡片**：点击右下角"+"按钮
- **搜索**：在搜索框中输入关键词
- **分类筛选**：点击分类标签切换显示

### 管理模式
- 点击右上角设置按钮进入管理模式
- 支持批量选择、删除、导入导出
- 可自定义布局和壁纸
